Welcome to the Site

This page is a demonstration of responsive web design. The layout will adapt to fit the screen size of your device, whether it's a mobile phone, tablet, or desktop. It also includes a special style for printing.

About Graceful Degradation

Graceful degradation means the content is still usable if the browser doesn't support the latest CSS features or if CSS is disabled entirely. This page is built with that principle in mind, using a mobile-first approach for core readability.

Feature Detection with @supports

The `@supports` rule is used to check for browser feature support before applying a specific style. For example, the `grid-template-columns` property is only applied if the browser supports CSS Grid. This ensures that the layout will not break on older browsers.
