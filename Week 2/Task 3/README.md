This project demonstrates **robust data fetching in React with TypeScript + Vite**, focusing on:  
- **Request cancellation** with `AbortController`.  
- **Deduping requests** so the same request is not made multiple times.  
- **Shared fetch cache** for efficiency.  
- **Clean unmount behavior** to prevent memory leaks.  

## Deliverables  
- **Fetch module** (`src/utils/fetchClient.ts`) implementing cancellation, deduping, and caching.  
- **Examples** (Todo component + log demo showing dedupe behavior).  
- **Tests** validating fetch logic and component behavior.  

## Acceptance Criteria  
- No memory leaks.  
- Aborted requests are handled gracefully.  
- Requests are deduped and cached correctly.  
- All tests pass.
